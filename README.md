This repository contains a "quick and dirty" analysis of package versions in the current Bioconductor devel branch, compared to the package versions provided in the corresponding GitHub repository. The motivation to look into this came from a question raised by Laurent Gatto in his talk at the EuroBioC2026 conference. At this point, the analysis only considers packages with a 'github.com' URL provided in the URL field in the DESCRIPTION file.
